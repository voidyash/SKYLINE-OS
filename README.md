# 🖥️ DevOS — Browser-Based Operating System UI

A fully interactive, browser-based operating system built using **pure HTML and CSS** — no JavaScript.

This project simulates a desktop environment with multiple apps, smooth transitions, and a cohesive system UI, demonstrating how far CSS can be pushed for interaction and state management.

---

## 🚀 Features

* 🧠 **CSS-Driven State Management**

  * Uses `:target` and `:checked` for app switching and interactions

* 🪟 **Window-Based UI System**

  * Multiple apps open as windows with smooth animations

* 💻 **Interactive Terminal**

  * Command simulation using CSS radio inputs

* ⚙️ **Settings Panel**

  * Sidebar navigation with dynamic content switching

* 📁 **Project Showcase**

  * Structured UI for displaying work inside the OS environment

* 📅 **Calendar UI**

  * Clean grid-based calendar interface

* 🎮 **Game Placeholder**

  * Space for a CSS-based mini-game

* 🎨 **Custom UI Design**

  * Dark theme, glassmorphism effects, and modern layout

* 📱 **Responsive Design**

  * Adapts for both desktop and mobile layouts

* ✨ **Micro Interactions**

  * Hover effects, transitions, and subtle animations

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

  * Flexbox & Grid
  * CSS Variables
  * Keyframe Animations
  * Pseudo-classes (`:target`, `:checked`)
---

## 🧩 How It Works

This project avoids JavaScript entirely by using CSS-based logic:

* `:target` → Opens and closes app windows
* `:checked` → Handles tab switching (Settings, Terminal)
* Transforms & animations → Simulate real UI behavior

The result is a fully interactive system using only layout and styling logic.

Built as part of a competitive web development challenge where the goal was to create **high-impact UI using only HTML and CSS**.

* No drag-and-drop window movement
* Static data (simulated interactions)
